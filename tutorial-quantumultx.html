Quantumult X
使用教程

Quantumult X（圈 X）是 iOS 端功能最强的代理工具，支持策略组、脚本复写、广告过滤、流媒体解锁分流。上手比小火箭复杂，但值得学。

购买与下载

⚠️ 非大陆区 Apple ID 才能下载，价格约 $7.99，比小火箭贵但功能更强。购买方式同 Shadowrocket，需美区/港区账号。
切换美区 Apple ID 下载
App Store → 头像 → 退出登录 → 用美区账号登录 → 搜索「Quantumult X」购买下载。下载完可切回国区账号。

添加订阅（节点）

打开圈 X，进入设置
打开 App，右下角点击「风车图标」进入配置页面。
添加节点订阅
找到「节点」→「引用（订阅）」→ 右上角「+」→ 粘贴机场提供的 Quantumult X 订阅链接 → 确认保存。
更新节点
在引用列表中，左滑条目点「更新」，或点右上角「全部更新」刷新所有订阅节点。

配置策略（分流规则）

圈 X 最大的亮点是策略组，可以针对不同 App/网站指定走不同节点。

使用机场提供的配置文件
大多数机场会提供 Quantumult X 专用配置文件链接（.conf 格式）。在圈 X「配置文件」中选择「引用配置文件」，粘贴链接导入，可省去手动配置策略组的步骤。
手动添加分流规则（可选）
「规则」→「引用规则」→ 粘贴第三方规则集链接（如 blackmatrix7 的规则集），可实现 Netflix、Disney+ 等流媒体自动分流。

开启连接

选择节点
回到主页 → 点击中央圆形区域 → 进入节点列表，选择延迟低的节点（长按可测速）。
开启代理
回到主页点击右上角的「连接开关」→ 允许 VPN 权限 → 图标变亮即为连接成功。
脚本复写：圈 X 支持 JS 脚本，可用于 App 去广告、解锁会员等。可在「复写」→「引用复写」中添加第三方脚本订阅（需自行评估风险）。
