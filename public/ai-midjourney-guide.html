Midjourney国内用户完整使用指南：注册、出图、订阅省钱技巧

从Discord注册到第一张图，再到省着用不超支的完整路径

Midjourney是什么，为什么它还值得用

在AI绘图这条赛道上，Midjourney依然是2026年最受专业设计师和创意人信赖的工具之一。它不靠开源模型堆砌，而是由一支精小的独立团队持续打磨，生成风格具有强烈的辨识度——光影层次丰富、构图克制、细节密度高。无论是商业插画、概念原画、品牌视觉，还是个人创作探索，它都有独到之处。

对国内用户来说，使用Midjourney有两道门槛：一是需要科学上网，二是整个服务架构在Discord之上，初次接触会觉得绕。本文把这两个问题一并拆解，带你从零走完注册、生图、订阅和省钱的完整流程。

国内使用Midjourney需要全程稳定的国际网络连接。建议使用延迟低于100ms的节点，否则Discord加载缓慢，图片生成指令容易超时。

第一步：注册Discord账号

Midjourney没有独立客户端，所有交互都通过Discord进行。如果你还没有Discord账号，先去 discord.com 完成注册。建议使用Gmail或Outlook邮箱，QQ邮箱有时会收不到验证邮件。

注册流程：打开discord.com，点击「Register」，填写用户名、邮箱和密码，完成邮件验证。验证完成后，Discord会引导你创建或加入服务器，这一步可以跳过，直接进入下一步。

Discord用户名建议使用英文加数字组合，避免使用特殊字符，后续和Bot交互时更顺畅。账号注册完成后，先在设置里开启双重验证（2FA），防止账号被盗用。

账号注册好之后，前往 midjourney.com，点击右下角「Sign In」，用Discord账号授权登录。首次登录会跳转回Discord，并自动将你加入Midjourney官方服务器。

第二步：在Discord里生成第一张图

进入Midjourney官方Discord服务器后，左侧频道列表会看到若干「#newbies-」开头的频道，这些是新手公共频道。点击任意一个进入，在底部输入框输入 /imagine，Discord会自动弹出指令补全，选择 /imagine prompt，然后在 prompt 后面输入你的描述词，回车发送。

例如：/imagine prompt a misty mountain village at dawn, ink wash painting style, soft morning light, highly detailed

发送后，Midjourney Bot会在频道中回复一条消息，先显示进度条，大约30到60秒后生成一组4张预览图。图片下方有U1-U4（放大单张）和V1-V4（基于某张变体衍生）两排按钮，U是Upscale的缩写，点击即可获得高分辨率单图。

公共频道里所有人的生成结果都公开可见，如果想私密生图，需要订阅Standard或Pro以上方案，并在个人DM频道与Midjourney Bot对话。

核心指令速查

掌握以下几条指令，能覆盖90%的日常使用场景：

  • /imagine prompt [描述] — 文字生图，最基础的指令
  • /settings — 打开设置面板，切换模型版本、画质、风格化强度
  • /info — 查看当前账号的剩余GPU时间和订阅状态
  • /blend — 上传2到5张图片，融合风格生成新图
  • /describe — 上传一张图，让AI反推可能的Prompt描述
  • /prefer option set — 保存常用参数组合，避免每次重复输入
  • --ar 16:9 — 指定画幅比例，支持任意比例
  • --style raw — 关闭Midjourney默认美化，输出更接近Prompt原意
  • --no [元素] — 排除不想要的内容，例如 --no text, watermark
  • --seed [数字] — 锁定随机种子，便于复现或微调结果

Fast模式和Relax模式的区别

这是理解Midjourney订阅价值的关键。Fast模式使用专用GPU资源，任务立即进入队列，通常30到60秒出图。Fast时间是每月配额制，用完即止，追加购买按$4/小时计费。

Relax模式（Standard及以上）使用空闲GPU资源，排队时间从几十秒到几分钟不等，高峰期可能更长，但不消耗Fast配额，相当于无限生图。对于不赶时间的探索性创作，Relax模式完全够用。

Turbo模式（Pro/Mega）速度是Fast的两倍，但消耗两倍Fast时间，适合需要快速迭代的专业场景。

在/settings里把默认模式设为Relax，只在需要快速出结果时手动切到Fast。这一个习惯能把Fast配额的有效使用周期延长2到3倍。

省钱技巧：让订阅费用减半

Midjourney的订阅费用对国内用户不算便宜，但有几个经过验证的方法能显著降低实际成本。

1. 年付立省20%。如果你确定会持续使用超过三个月，直接年付，Standard年付$288比月付$360便宜了$72，相当于免费多用两个多月。

2. 用Relax模式打底，Fast只做精修。大多数创意探索不需要Fast速度，Relax完全够。把Fast时间留给最终稿的Upscale和微调变体，而不是前期大量测试Prompt。

3. 善用/prefer option set保存Prompt片段。固定风格关键词、常用参数（如--ar、--style、--q）保存成一个option，每次生图只需调用，减少反复试验次数，节省GPU时间。

4. 按需升降级，不要长期持有高档方案。接到大项目时升到Pro，项目结束立即降回Standard。Midjourney支持在当前计费周期结束时自动降级，不会浪费已付费用。

5. 用/describe反推参考图的Prompt，再二次创作。这比从零摸索Prompt效率高得多，能大幅减少无效生图消耗。

6. 充分利用Vary Region（局部重绘）。当一张图整体满意但局部有瑕疵时，用Vary Region只重绘问题区域，而不是整张图重新生成，节省Fast时间。

Midjourney不支持退款，订阅后当月费用不可退还。建议先以月付方式试用一两个月，确认使用习惯后再考虑年付锁定优惠。

国内支付方式

Midjourney只接受国际信用卡（Visa、Mastercard、American Express）和部分虚拟卡。国内用户常用的方案有：招商银行或工商银行的全币种信用卡（Visa/Mastercard通道）、Depay或Wise虚拟卡、或通过家人在海外的信用卡代付。

支付宝和微信支付目前不被直接支持。如果你有美区Apple ID，可以通过App Store的礼品卡充值后订阅iOS版，但Midjourney暂无独立App，此路不通。最稳妥的方案是申请一张支持国际支付的双币信用卡。

Depay等虚拟卡开卡需要KYC实名认证，充值需要USDT，整体流程对新手有一定门槛。如果只是想试用，建议先向有境外卡的朋友借刷，再决定是否自己办卡。

常见问题排查

Discord频道刷新慢、图片加载不出来：多半是节点问题，换一个延迟更低的节点，或切换到全局代理模式。

/imagine指令无响应：确认Bot是否在当前频道激活，可以输入 @Midjourney Bot 测试。公共服务器高峰期Bot偶尔会积压，稍等片刻重试。

生成图片被内容过滤拦截：Midjourney对部分关键词有屏蔽机制，调整描述方式，避免使用被标记的词汇，换成更迂回的表达通常能绕过。

Fast时间提前耗尽：先检查/info确认剩余时间，切换到Relax模式继续使用，等下个计费周期重置，或按$4/小时追购Fast时间。

Upscale后图片分辨率不够用：使用Upscale (2x)或Upscale (4x)获得更高分辨率版本，Pro方案支持更高倍率放大。
