V2RayU
macOS 使用教程

V2RayU 是 macOS 上的 v2ray 图形客户端，常驻菜单栏，操作简单，适合不想折腾但又需要 v2ray 协议支持的用户。推荐作为 ClashX 的备用工具。

下载安装

下载 V2RayU
前往 GitHub 搜索「V2RayU」，在 Releases 页下载最新版 .dmg 文件。
安装并放行
打开 dmg，将 V2RayU 拖入 Applications。首次运行如遇「无法验证开发者」提示，前往「系统偏好设置 → 安全性与隐私 → 通用」点击「仍要打开」。

添加订阅 / 节点

通过订阅链接导入（推荐）
菜单栏 V2RayU 图标 → 「订阅设置」→ 粘贴机场提供的 v2ray 订阅链接 → 点「更新订阅」下载节点列表。
手动添加单个节点
菜单栏 → 「服务器设置」→ 点「+」→ 选择协议类型（VMess / Shadowsocks 等）→ 填入服务器地址、端口、密码等信息 → 保存。
选择节点并连接
菜单栏 → 选择节点 → 点「打开 V2Ray」→ 再点「HTTP 代理开启」即可开始使用。
代理模式：V2RayU 提供「自动模式」（PAC 分流）和「全局模式」，日常建议使用自动模式，国内网站直连，国外走代理。
注意：V2RayU 更新频率较低，部分较新的协议（如 VLESS + Reality）支持有限。有更高需求建议改用 ClashX Pro。
