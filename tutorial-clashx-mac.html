macOS 2023-08-10

ClashX / ClashX Pro
使用教程

ClashX 是 macOS 端最简洁的 Clash 客户端，常驻菜单栏，一键切换代理模式。ClashX Pro 在此基础上增加了 TUN 模式，推荐直接用 Pro 版。

下载安装

下载 ClashX Pro
在 GitHub 搜索「ClashX Pro」，下载最新版 .dmg 文件。也可从机场的「客户端下载」页获取链接。
安装与授权
打开 dmg，将 ClashX Pro 拖入 Applications 文件夹。首次启动时，系统会询问是否安装帮助程序（Helper），点击「Install」并输入 Mac 密码授权。
找到菜单栏图标
安装完成后，屏幕右上角菜单栏会出现猫咪图标，所有操作都从这里进行。

导入订阅

复制机场订阅链接
登录机场控制台，复制「Clash 订阅链接」或「一键导入」地址。
在 ClashX Pro 中添加
点击菜单栏猫咪图标 → 「Config」→「Remote Config」→「Manage」→ 点击「Add」→ 粘贴订阅链接 → 确认。
选中配置
在 Remote Config 列表中，点击刚添加的配置右边「Select」使其生效，图标变为选中状态。

开启代理

开启系统代理
点击菜单栏猫咪 → 勾选「Set as System Proxy」，此时 Safari、Chrome 等浏览器即可访问外网。
选择代理模式
菜单栏 → 「Proxy Mode」：
· Rule：规则分流，日常推荐
· Global：全部走代理
· Direct：全部直连（等于关闭）
（进阶）开启 TUN 模式
菜单栏 → 勾选「Enhanced Mode (TUN)」，可让终端、游戏等不走系统代理的程序也通过 Clash 分流。需要先安装 Helper。
更新订阅：菜单栏 → Config → Remote Config → 右键配置 → 「Update」。建议配置自动更新间隔（单位：小时）。
⚠️ macOS 隐私提示：如果系统弹出「无法打开，因为无法验证开发者」，前往「系统偏好设置 → 安全性与隐私」点击「仍要打开」。
