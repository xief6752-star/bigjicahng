Clash for Windows
使用教程

Clash for Windows（CFW）是目前 Windows 端功能最完整的代理客户端，支持订阅导入、规则分流、多种协议。本教程从零开始，十分钟内完成配置。

第一步：下载安装

下载 CFW
前往 GitHub 搜索「Clash for Windows」或从机场面板的「客户端下载」页获取。下载 Clash.for.Windows.Setup.x.x.x.exe 安装包。
安装程序
双击安装包，默认安装路径即可。安装完成后桌面会出现 Clash 图标，双击打开。

第二步：导入订阅链接

复制机场订阅链接
登录你的机场控制台，找到「订阅链接」或「一键导入 Clash」，复制链接地址。
在 CFW 中添加
打开 CFW，点击左侧「Profiles」（配置文件）→ 在顶部输入框粘贴订阅链接 → 点击「Download」下载配置。
激活配置
下载完成后，点击配置文件卡片将其选中（出现蓝色高亮边框即为激活）。
订阅更新：节点有变化时，回到 Profiles 页，右键点击配置文件 → 选择「Update」即可更新订阅。建议开启自动更新。

第三步：选择代理模式

点击左侧「Proxies」进入节点选择页，顶部有三种模式：

Rule（规则模式）—— 推荐日常使用
国内网站走直连，国外网站自动走代理。绝大多数情况下选这个。
Global（全局模式）
所有流量都走代理。访问国内网站会变慢，一般用于排查问题时临时切换。
Direct（直连模式）
所有流量直连，不走代理。等同于关闭 Clash，用于临时关闭时。

第四步：开启系统代理

回到 CFW 主界面（General 页），打开「System Proxy」开关，此时浏览器已可正常访问被墙网站。

⚠️ 注意：部分情况下 CFW 关闭后系统代理未自动关闭，导致断网。解决方法：关闭 CFW 前先手动关闭「System Proxy」开关，或在系统设置里手动关闭代理。

常见问题

订阅下载失败
检查订阅链接是否正确复制（注意不要有空格），或尝试将 System Proxy 先关闭再下载。
开启后浏览器还是无法访问
确认 System Proxy 已开启，且选择了 Rule 或 Global 模式，节点已选择（不是 DIRECT）。
游戏不走代理
Rule 模式下大多数游戏走直连，如需代理游戏流量，改为 Global 模式或配置 TUN 模式。
