Shadowrocket（小火箭）
使用教程

Shadowrocket 是 iOS 端使用最广泛的代理客户端，¥18 买断，支持多种协议和订阅格式。本教程从购买账号到配置完成，全程图文说明。

购买 Shadowrocket

⚠️ 必须使用非大陆区 Apple ID
Shadowrocket 在中国区 App Store 已下架，需要美区/港区/台区账号才能购买和下载。如果没有，需要先注册一个或购买共享账号。
切换 App Store 区域
打开 App Store → 点击右上角头像 → 拉到底部「退出登录」→ 使用美区 Apple ID 登录。
搜索并购买
搜索「Shadowrocket」，价格 $2.99（约 ¥18），一次购买终身使用。购买后可切换回国区 Apple ID，App 保留。
省钱方式：可以在机场面板或论坛找「美区 Apple ID 共享账号」，仅用来下载 App，下载完立即切回自己账号。切勿在共享账号里开启「查找我的 iPhone」。

添加订阅链接

复制机场订阅链接
登录机场控制台，找到「订阅链接」或「一键导入」，复制适用于 Shadowrocket / SS / VMess 的链接。
在 Shadowrocket 中添加
打开 Shadowrocket → 点击右上角「+」→ 类型选「Subscribe」→ 粘贴订阅链接 → 右上角「完成」。
更新订阅
在订阅条目上左滑 → 点「刷新」，或下拉节点列表触发更新。建议开启自动更新（设置 → 服务器订阅 → 自动更新）。

选择节点 & 连接

选择节点
在节点列表中点击一个节点，左边出现对勾即为选中。可以点右上角的信号图标测速，选延迟低的。
开启连接
回到首页，点击顶部的「未连接」开关 → 系统弹出 VPN 权限请求 → 允许 → 开关变绿色即为已连接。

代理模式说明

首页顶部「全局路由」可切换模式：

代理（Proxy）—— 日常推荐
根据规则自动判断，国内直连、国外走代理。速度和体验最均衡。
直连（Direct）
所有流量直连，不走代理，等同于关闭。
全局（Global）
所有流量走代理，访问国内网站会变慢，排查问题时使用。
