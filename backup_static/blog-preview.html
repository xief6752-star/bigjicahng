写的博客系统

基于 Koa2 构建 REST API，配套 Vue/React 管理后台和 Next.js/Nuxt.js 前端。 代码结构清晰，适合 Node.js 入门学习，也可作为毕设项目基础。

后端 Node.js + Koa2
数据库 MySQL
启动端口 localhost:5000
作者 LFB
功能模块

系统实现了一个博客所需的完整后端接口，包括文章管理、分类管理、管理员认证， 以及工程层面的日志、参数校验、API 文档和单元测试，是一套可以直接拿来参考的工程化实践。

01
管理员认证
JWT 无状态登录，token 校验中间件保护所有写操作接口。
02
文章 & 分类 CRUD
完整增删改查，支持 Markdown 内容存储，文章分页列表，草稿/发布状态管理。
03
图片上传
封面图上传接口，存储至本地文件系统，返回可访问 URL。
04
参数校验 & 统一错误处理
请求参数自动校验中间件，错误信息格式统一，前端对接省心。
05
Swagger API 文档 & 单元测试
自动生成接口文档，核心接口有测试用例，工程质量有保障。
技术栈
Koa2 后端核心框架，洋葱模型中间件，处理路由与请求响应
Sequelize ORM 层，操作 MySQL，Model 定义与数据迁移
MySQL 关系型数据库，存储文章、分类、用户数据
jsonwebtoken JWT 生成与验证，实现无状态管理员认证
log4js 日志记录，请求日志 + 错误日志分级输出
koa-swagger 自动生成 Swagger API 文档
mocha / chai 单元测试框架，接口测试用例
PM2 生产环境进程管理与守护
项目结构
nodejs-koa-blog/ 
app/
  ├── api/          # 路由层 — 文章、分类、用户接口
  ├── dao/          # 数据访问层 — 封装数据库查询
  ├── models/       # Sequelize Model 定义
  ├── validators/   # 参数校验规则
  ├── middleware/   # JWT 鉴权、错误处理等中间件
  └── lib/          # 工具函数
config/             # 数据库连接、密钥等配置
logs/               # 运行日志
test/               # 单元测试
.env.development    # 本地环境变量
app.js              # 入口文件
快速开始
克隆仓库
git clone https://github.com/lfb/nodejs-koa-blog.git
cd nodejs-koa-blog
创建数据库
需要本地安装 MySQL，然后创建数据库：
mysql -uroot -p你的密码

CREATE DATABASE IF NOT EXISTS boblog
  DEFAULT CHARACTER SET utf8mb4;
配置环境变量
修改根目录 .env.development：
.env.development
DB_NAME     = 'boblog'
DB_HOST     = 'localhost'
DB_PORT     = 3306
DB_USER     = 'root'
DB_PASSWORD = '你的密码'
安装依赖 & 启动
npm install
npm run dev

# 看到这行说明启动成功 👇
Server running at http://localhost:5000
配套子项目

后端 API 是核心，前端可以按自己熟悉的技术栈选一个搭配使用。
