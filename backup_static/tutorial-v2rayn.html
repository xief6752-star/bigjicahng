v2rayN
使用教程

v2rayN 是 Windows 端轻量级的 v2ray 图形客户端，界面简洁，支持 VMess、VLESS、Trojan、Shadowsocks 等协议。适合作为 Clash 的备用客户端。

下载安装

下载 v2rayN
前往 GitHub 搜索「v2rayN」，在 Releases 页下载最新版 v2rayN-With-Core.zip（含核心文件）。
解压运行
解压到任意文件夹，双击 v2rayN.exe 运行，程序会在任务栏右下角托盘区显示图标。
⚠️ 需要 .NET 6.0 运行环境：如启动报错，请前往微软官网下载安装 .NET Desktop Runtime 6.0 或更高版本。

添加订阅

打开订阅管理
在 v2rayN 主窗口顶部菜单栏点击「订阅分组」→「订阅分组设置」。
添加订阅链接
点击「+」添加新订阅 → 在「地址」栏粘贴机场订阅链接（选择 v2ray/通用格式）→ 点「确定」保存。
更新订阅
回到主界面，点「订阅分组」→「更新全部订阅（不通过代理）」，节点列表会刷新显示所有节点。

选择节点 & 开启代理

选择节点
在节点列表中双击一个节点设为当前使用。可按 Ctrl+Enter 对选中节点测速，选延迟最低的。
开启系统代理
右下角托盘图标右键 → 「系统代理」→ 选「自动配置系统代理」，浏览器即可访问外网。
选择路由模式
主窗口底部选择路由方案：
· 绕过大陆（推荐）：国内直连，国外走代理
· 全局：所有流量走代理
TUN 模式：如需让终端/游戏也走代理，可在设置中开启 TUN 模式，需要管理员权限。
